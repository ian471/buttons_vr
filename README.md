## Enable WebXR Local Testing

1. In Chrome (or Oculus Browser), go to `chrome://flags/#unsafely-treat-insecure-origin-as-secure`
1. Select **Enabled** and add your `http://<host>:8080` origin.
1. Restart the browser.
